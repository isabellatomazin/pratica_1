Praticando com o Git e Github

O que é o GIT?

O GIT é um sistema de controle de versão, utilizado para criar um histórico de alterações no código de projetos.

O que é o Github

GitHub é um repositório remoto, criado com o serviço do Git funciona como um servidor que agrega todas as modificações realizadas por cada um dos membros envolvidos em um projeto. Além de poder ser, também, uma espécie de "rede social" para programadores.
